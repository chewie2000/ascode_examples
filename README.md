# As-Code Examples

Demo area for Sigma Computing workbook-as-code and data model-as-code demonstrations.

## Structure

- `models-as-code/` — Sigma data model spec JSON files that can be deployed via the Sigma REST API

## Usage

Model specs can be deployed to Sigma using `PUT /v2/dataModels/<dataModelId>/spec` or created with `POST /v2/dataModels/spec`.
